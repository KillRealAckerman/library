# Описание
Фласк‑приложение для библиотеки: авторизация с ролями, работа с SQL-запросами по книгам/поставкам, создание и просмотр месячных отчётов, и оформление поставок через корзину. Данные берутся из MySQL, есть кэширование Redis для списка книг. Роль admin/Librarian/Secretary определяет, что видно: запросы, отчёты или корзина поставок.
